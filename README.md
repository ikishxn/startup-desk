Startup Desk

A modern campus startup showcase platform where students can present startup ideas, explore innovative projects, connect with collaborators, and build entrepreneurial communities.

Features
User authentication with Firebase
Startup idea showcase system
Upload startup images and media using Cloudinary
Responsive React UI
Real-time database integration with Firebase Firestore
Secure environment variable configuration
Fast and optimized Vite setup

Tech Stack
Frontend
React.js
Vite
Tailwind CSS
JavaScript
Backend & Services
Firebase Authentication
Firebase Firestore
Firebase Storage
Cloudinary


Installation

Clone the repository:git clone https://github.com/ikishxn/startup-desk.git

Go to the project folder:cd startup-desk
npm install
npm run dev
